- Project random: `swift run forge project --random [tier] [layer]`

## Flow Notes
- Default flow gates pass-level projects after each challenge block: Core 1, Core 2, Core 3, Mantle 1, Mantle 2, Mantle 3, Crust 1, Crust 2, Crust 3.
- Each stage includes a deterministic review gate (three challenges, repeated twice) before its project.
- Bridge challenges run after Core 3 and Mantle 3 to smooth transitions.
- Extra challenges and projects are intended for optional practice and are used by random mode or manual progress jumps.

## Assessment
- The curriculum progresses from syntax fundamentals to practical data manipulation, then to types, generics, and advanced language/runtime topics.
- Core introduces concepts before integration challenges; Mantle reinforces structure and abstraction; Crust adds concurrency, advanced Swift features, and professional practices.
- Practice coverage is strong: extras provide reinforcement without disrupting the main flow.
- Manual-check items are isolated to topics that require external tooling or human verification.
